### 关于每个文件夹中另外出现的 Dev-Cpp-5_11.cpp文件的说明
Dev-C++ 5.11 的 UTF-8 支持存在问题，其它 IDE（包括 Embarcadero 的 Dev-C++ 6.3）编写的含中文的程序在 Dev-C++ 5.11 会出现乱码，而 Dev-C++ 5.11 写的程序在其它 IDE 中也会乱码。
